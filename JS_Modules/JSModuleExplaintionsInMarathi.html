JavaScript Module Concept

Importance of JavaScript Modules

JavaScript modules allow you to divide your code into separate files, making it more organized and manageable. You can export variables and functions from one module and import them into another. This helps in code reusability and better maintenance.

Use of the Export

Variable Export

Variables can be exported from a module so they can be used in other files.

    // variable.js
    export const pi = 3.14;

Function Export

Functions can also be exported and imported, allowing you to keep your code modular.

    // mathFunctions.js
    export function add(a, b) {
        return a + b;
    }

Example of Using Exports

Here’s how you can import the exported variable and function in another file:

    // main.js
    import { pi } from './variable.js';
    import { add } from './mathFunctions.js';

    console.log(`Value of Pi: ${pi}`);
    console.log(`Sum: ${add(2, 3)}`);

JavaScript मॉड्यूलचे महत्त्व

JavaScript मॉड्यूल तुम्हाला तुमचा कोड वेगवेगळ्या फाइल्समध्ये विभाजित करण्यास मदत करतात, ज्यामुळे तो अधिक संघटित आणि व्यवस्थापित करणे सोपे होते. तुम्ही एक मॉड्यूलमधून व्हेरिएबल्स आणि कार्ये निर्यात करू शकता आणि दुसऱ्या मॉड्यूलमध्ये त्यांचा आयात करू शकता. यामुळे कोडचे पुनर्वापर आणि चांगली देखभाल होऊ शकते.

निर्यात वापरणे

व्हेरिएबल निर्यात

व्हेरिएबल्स एक मॉड्यूलमधून निर्यात केल्या जाऊ शकतात जेणेकरून त्या इतर फाइल्समध्ये वापरल्या जाऊ शकतील.

    // variable.js
    export const pi = 3.14;

कार्य निर्यात

कार्ये देखील निर्यात केली जाऊ शकतात आणि आयात केल्या जाऊ शकतात, ज्यामुळे तुमचा कोड मॉड्यूलर राहतो.

    // mathFunctions.js
    export function add(a, b) {
        return a + b;
    }

निर्यात केलेल्या कार्यांचा वापर

इतर फाइलमध्ये निर्यात केलेली व्हेरिएबल आणि कार्ये कशाप्रकारे आयात करू शकता याचे उदाहरण:

    // main.js
    import { pi } from './variable.js';
    import { add } from './mathFunctions.js';

    console.log(`Pi चा मूल्य: ${pi}`);
    console.log(`योग: ${add(2, 3)}`);
