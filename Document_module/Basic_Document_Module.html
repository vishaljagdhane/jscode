HTML DOM (Document Object Model)

Introduction

The HTML DOM is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programming languages, such as JavaScript, to manipulate the content, structure, and style of web pages.

Key Concepts

  • Document: The entire web page, represented as an object.
  • Element: An individual component of the page (e.g., <p>, <div>).
  • Node: The basic unit of the DOM, which can be an element, attribute, or text.
  • Tree Structure: The DOM is structured as a tree, with the document as the root node and elements as child nodes.

Manipulating the DOM with JavaScript

JavaScript can be used to change the document structure, style, and content dynamically. Below are some common methods:

Accessing Elements

const element = document.getElementById('myElement'); // Access by ID
const elements = document.getElementsByClassName('myClass'); // Access by class
const queryElement = document.querySelector('.myClass'); // Access using CSS selectors

Modifying Content

element.innerHTML = 'New Content'; // Change HTML content
element.textContent = 'New Text'; // Change text content

Changing Styles

element.style.color = 'blue'; // Change text color
element.style.display = 'none'; // Hide the element

Adding and Removing Elements

// Create a new element
const newElement = document.createElement('div');
newElement.textContent = 'This is a new element';
document.body.appendChild(newElement); // Add to the body

// Remove an element
document.body.removeChild(element); // Remove an element from the body

Example: Simple DOM Manipulation

<!DOCTYPE html>
<html lang="en">
<head>
    <title>DOM Example</title>
</head>
<body>
    <h1 id="header">Hello, World!</h1>
    <button id="changeButton">Change Text</button>

    <script>
        document.getElementById('changeButton').onclick = function() {
            document.getElementById('header').textContent = 'Text Changed!';
        };
    </script>
</body>
</html>

Conclusion

The HTML DOM provides a powerful interface for interacting with web pages. Understanding the DOM and how to manipulate it with JavaScript is essential for creating dynamic and interactive web applications.
