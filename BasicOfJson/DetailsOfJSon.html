Introduction to JSON

What is JSON?

JSON stands for JavaScript Object Notation. It is a lightweight data interchange format that is easy for humans to read and write and easy for machines to parse and generate.

Why Use JSON?

  • Human-readable format: Easy to understand and edit.
  • Language-independent: JSON is text and can be used with most programming languages.
  • Lightweight: Ideal for data transmission over networks.

Basic Structure of JSON

JSON data is represented in key-value pairs, similar to JavaScript objects. Here’s a basic example:

{
    "name": "Vishal",
    "age": 30,
    "city": "Aurangabad"
}

JSON Array Example

JSON can also contain arrays. Here’s an example of a JSON array:

[
    {
        "name": "Vishal",
        "age": 30
    },
    {
        "name": "Haribhau",
        "age": 35
    }
]

How to Use JSON in JavaScript

You can parse JSON data and convert it into a JavaScript object using JSON.parse() and convert a JavaScript object into JSON using JSON.stringify().

Example:

const jsonString = '{"name":"Vishal","age":30}';
const jsonObject = JSON.parse(jsonString); // Converts JSON string to object

console.log(jsonObject.name); // Output: Vishal

const newJsonString = JSON.stringify(jsonObject); // Converts object to JSON string
console.log(newJsonString); // Output: {"name":"Vishal","age":30}
